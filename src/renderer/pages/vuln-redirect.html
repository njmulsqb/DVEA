Open Redirect via Deep Link

Vulnerability: This demo shows how an Electron app can be vulnerable to open redirects when handling deep links. If a deep link contains a redirect URL, the app will redirect to it without validation, allowing attackers to send users to malicious sites.

How to invoke: Use a deep link in the format dvea://?redirect=https://example.com to trigger the redirect. The app will automatically redirect to the specified URL when opened via this deep link.

Example: dvea://?redirect=https://evil.com
