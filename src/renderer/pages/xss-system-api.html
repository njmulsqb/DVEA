XSS: Overprivileged ContextBridge

XSS with direct access to shell commands via a preload-exposed API

✓ NodeIntegration: false ✓ ContextIsolation: true ✗ Sandbox: false

Guide: What is happening?

The renderer has access to privileged APIs via a custom preload script that exposes window.systemAPI, even though nodeIntegration is false and contextIsolation is true. XSS in the renderer can call window.systemAPI.runCommand() to execute arbitrary shell commands.

Example payload: 
<img src=x onerror="window.systemAPI.runCommand('open -a Calculator')">
Runs Calculator on macOS. Substitute a command appropriate for your OS.
Impact: Full system compromise — read/write files, execute OS commands, exfiltrate data. This demonstrates why least privilege is critical when designing contextBridge APIs, even when Electron's core security features are enabled.
