XSS: System APIs Exposed

This page demonstrates an XSS vulnerability in the renderer process with privileged Node.js or Electron APIs exposed.

Explanation: This is a reflected XSS vulnerability, but in this case, the renderer has access to Node.js or Electron APIs (such as require('fs') or window.electron). An attacker can execute arbitrary JavaScript and also access the file system, OS, or main process APIs, leading to full system compromise.
Impact: The attacker can go beyond browser-like attacks and perform actions like reading/writing files, executing system commands, or exfiltrating sensitive data from the user's machine.
